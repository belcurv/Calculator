ANGULAR CALCULATOR

This is a simple calculator made with javascript, Angular 1.x, HTML and CSS flexboxes. Credit my man Pete for the calculator logic.

Highlights: the calculator and chatterbox are Angular directives; both rely of an Angular factory for logic. The chatterbox describes the state of the calculator after each button press.

It's served by a Node/Express backend but it doesn't really require one.
